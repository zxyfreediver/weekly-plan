# Project Brief (Persistent)

## 产品愿景

周记清单：个人周度任务管理工具，帮助职场人按周记录、分类、完成待办，并支持 AI 总结已完成工作。

## 编码约定

- **语言**: TypeScript
- **格式化**: Prettier
- **Lint**: ESLint
- **命名**: 组件 PascalCase，文件 kebab-case，API REST 风格
- **架构**: Route 薄层，逻辑在 services

## 质量门禁

- 提交前运行 `npm run lint`
- 有测试时运行 `npm test`
- 不提交 `any` 类型
- 不提交占位内容（Lorem ipsum）到生产

## 关键命令

- `npm run dev` — 开发
- `npm run build` — 构建
- `npm run lint` — 检查
- `npm test` — 测试

## 更新节奏

- 项目结构变化时更新本 brief
- 新增约定时补充到此文档
